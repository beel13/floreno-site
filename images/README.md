# FLORENO image library

Real source photography and specimen portraits, as individual files so design
tools can import them directly. (The site's own index.html has these inlined
as base64, which is why they cannot be pulled out of it.)

Raw URL pattern:
`https://raw.githubusercontent.com/beel13/floreno-site/master/images/<file>`
